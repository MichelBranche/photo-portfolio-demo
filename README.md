# 🎞️ Rubina Stradella — Portfolio Experience

![Preview](./preview.jpg)

> Brutalism, fotografia e sviluppo front-end si incontrano.

---

## ✨ Overview

Questo progetto è un portfolio fotografico sviluppato per Rubina Stradella, progettato per:

- trasmettere una forte identità visiva
- valorizzare il contenuto fotografico
- creare un'esperienza immersiva e cinematografica

L'interfaccia combina estetica **brutalist/editoriale** con una logica di navigazione fluida e performante.

---

## 🚀 Tech Stack

- **HTML5**
- **CSS3 (Responsive, Clamp, Layout avanzati)**
- **JavaScript ES Modules**
- **GSAP (Flip, Animazioni avanzate)**
- **Google Drive API (opzionale)**
- Nessun framework → **approccio performance-first**

---

## 🧠 Architettura

Il progetto è costruito come sito statico modulare:

- `FashionGallery` → cuore logico dell'app
- sistema multi-progetto (14 serie fotografiche)
- gestione dinamica immagini (local + Drive)
- layout differenziati per ogni progetto

📁 Struttura:
/js/modules → logica
/css/projects → styling per serie
/portfolio → configurazioni e dati
/media → immagini

## 🎨 UX / Design

- Griglia dinamica editoriale
- Zoom immagini con GSAP Flip
- Slideshow per ogni cella
- Layout diversi per ogni progetto
- Stile brutalist con texture e layering visivo

---

## ⚡ Performance

Ottimizzazioni avanzate:

- Lazy loading intelligente immagini
- Performance tier (device-based)
- Riduzione animazioni su dispositivi lenti
- Supporto `prefers-reduced-motion`
- Ottimizzazione canvas/audio

---

## 📱 Responsive

Breakpoint principale: **900px**

- Mobile → esperienza ottimizzata touch
- Desktop → layout immersivo editoriale
